Los cargos listados aquí aplican únicamente para {{project.name_to_show}}. Si este proyecto no tiene cargos propios se usarán los cargos globales como respaldo.

Grupo: define si el cargo es visible para el módulo Comercial (Público) o si se usa internamente en la escala (Privado).
Anticipo: valor fijo que se pagará cuando se liquide un anticipo para ese cargo (si la regla de negocio lo permite).
Asesor por defecto: quien se asignará automáticamente cuando se genere una escala si el cargo pertenece al grupo Privado.
Incluir por defecto: si está activo, el cargo (y su asesor por defecto) se agrega automáticamente a cada nueva escala del proyecto.
Activo/Inactivo: solo los cargos activos aparecen para crear o modificar escalas; los inactivos quedan registrados para histórico pero no se ofrecen en los cargos disponibles.
